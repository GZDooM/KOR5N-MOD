====== KOR5N MOD для WoTB v.0.1.1.2 ======

Включает в себя:
- озвучку экипажа от KorbenDallas Топ Стрелок
- звуки выстрелов орудий из WoT 1.0
- музыкальное сопровождение при загрузке в бой
- упрощенный стандартный прицел
